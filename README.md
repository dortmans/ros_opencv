# ros_opencv

ROS package to support "OpenCV in ROS" course.

## Installation

Installation of this package can be done as follows:

```
cd ~/catkin_ws/src
git clone --depth 1 https://github.com/dortmans/ros_opencv.git
chmod +x ros_opencv/scripts/*
cd ~/catkin_ws
catkin_make
```
